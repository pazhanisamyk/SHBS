How to run the Seminar Hall Booking System Project using PHP and MySQL

1. Find the project zip file named as SHBS.

2. Extract the file and copy SHBS folder.

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/Html).

4.Open PHPMyAdmin (http://localhost/phpmyadmin).

5. Create a database with the name of  obbs

6. Import obbs.sql file(given inside the zip package inside the SHBS ===> SHBS ===> Database Folder ).

7. Run the script http://localhost/SHBS/SHBS
